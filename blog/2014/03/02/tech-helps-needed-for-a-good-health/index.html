科技+健康生活环境

Mar 2nd, 2014 12:08 pm

居住在一个大城市里，对生活环境的获取和牺牲都是必然的，科技在这方面能帮到的并不多。

健康的生活环境，以下几点是必须的：

  1. 清新干净空气

    这个貌似非常困难，大环境看来，工业/汽车是最大的威胁。

    小环境看来，窗户还做不到只让空气进出，把灰尘和有害毒物挡在窗外或直接消化掉。

  2. 安静

    环境噪音主要来自汽车，这个也是大环境，暂时是无法消灭的。

    小环境来看，对付噪音污染，可以安装隔音玻璃窗，或者搬家，要不就塞住自己的耳朵，此外也别无他法。

  3. 黑夜

    大环境光污染更是没办法了，部分昼夜的城市的必然产物。

    小环境，拉遮光窗帘还是可以解决的。

  4. 干净安全的饮水和食品

    大环境下这个东西只能靠社会良心了。

    小环境，可以自己种，或者多花些钱和精力找安全和有良心的来源。

要对付这些环境对健康的威胁，所谓高速发达的科学技术还是无能为力的。确切地说，这些威胁都来自科技发展的副效应。

目前唯一能够解决所有这些问题的办法就是，搬到一个远离科技，有大片绿树隔离的村落。

或者，期待科技更进一步，提供更可行的避免过滤尘灰、空气污染、噪音和光污染、食品污染的技术方案。（其实希望并不大，看看北京突破500的PM2.5就知道了）
