  • Software: XZoom (下载/Download, 27.9 KB) 用法/Usage

    Portable: Screen zoomer, color picker; Alt + Space hide/show, right mouse click to hide; scroll to change zoom ratio, middle mouse click to reset default zoom ratio.

    绿色软件: 倍数及大小设置，屏幕取色; Alt + Space切换隐藏/显示，右键点击隐藏；滚轮改变放大倍数，中键恢复缺省放大倍数。

  • Software: Letup (下载/Download, 140 KB) 用法/Usage

    Portable: Lock up your screen when time to have a break！For your good health. (win2k/vista/win7) DO NOT run it when playing games :-)

    绿色软件: 定时锁屏提醒休息，守护健康。右下角密码唤醒功能。

  • Web App: Mindon Eka (访问/Launch, Visit)

    Portable: Memory your accounts on the web; Bookmarklet, or Chrome App / Extension; Mail backup feature, password safe reminder.

    绿色软件: Eka服务为了方便管理和访问网站注册的众多帐号，提供Chrome应用/扩展、书签形式，邮件备份，安全密码回忆特性。
