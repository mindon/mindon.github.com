关于家乐福，我们是不是被忽悠了?

May 1st, 2008 1:04 pm

BlogMS original blog key: 1002094810, blog id: airoff History stat: 浏览/评论：299/0 , 日期：2008年5月1日 13:04

关于家乐福，我确实也受网上传闻影响，有抵触的心理。不过某天看到凤凰卫视中文台提到的一个信息，我反思了一下——会不会被某些人利用了呢？

您敢说我们敏感的爱国心理不会被利用吗？

凤凰卫视一则新闻提到某些不良组织威胁一些大企业的事情。我们太容易相信传闻了，也不管是否有证据，只要导致的行为看起来非常像爱国的行为就可以了，谁在乎是否传闻是否属实，又或者是否有真凭实据？假若情况如以下场景，那么这次家乐福的事情，我们的爱国心理会不会是被利用了呢？会不会被某些人忽悠了呢？

参与角色：

B 某些居心不良组织

Z 敏感爱国国人

C 家乐福

场景：

B 要求 C 提供资金支持，于是可能发生情况有两种：

1) C 提供支持，B 高兴；

2) C 不提供支持, B 很不高兴。

如果是情况1) B的行动一如既往；如果是情况2)，B 很不高兴——不支持？那就让 C 生意不好做——这太容易做到了：只需要在中国的网络上放出一条消息。

于是 Z 看到了这条消息，当然 Z 没法子验证消息的真实与否，不过还是信其无不如信其有——于是就传开了，一传十，十传百…… C 的话反而没人信了，生意也遭受损失…同时遭受损失的还有 Z 的同胞们。

嘿嘿，正中 B 的下怀：C 你不支持我 B，这就是下场！

于是下一次 B 要求 C 提供支持的时候 ……

假如这次家乐福事件真是类似场景，我们岂不是被人离间，被人忽悠了？

难道我们真的不管三七二十一，只要满足了我们的爱国主义表达就足够了吗？

相关报道

路透北京4月30日电—世界第二大零售商–法国家乐福(Carrefour)中国区副总裁贝多拉周二表示,因为理解中国人民的爱国热情和现在的心情,公司将取消”五一”期间中国地区所有促销宣传活动.

新华社周二晚间报导援引他的话称,作为一家大型企业,家乐福不愿扮演任何政治角色,只想做好自己分内的事.家乐福会一如既往地支持北京奥运会,为奥运做出贡献.

他同时否认家乐福支持过包括”藏独”活动在内的任何非法活动,并重申发生在巴黎的针对奥运圣火的攻击是完全不能接受和让人感到耻辱的.

此前,因奥运圣火在法国巴黎传递期间遭遇诸多阻挠,引发中国民众对法国政府强烈不满,国内部分城市发生民众抵制到家乐福购买商品的示威行为.

家乐福1995年进入中国市场,目前在中国雇用四万多人,占全部员工的99%,年销售额近300亿元人民币,所销产品95%由中国制造.
