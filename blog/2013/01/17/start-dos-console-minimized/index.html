Start DOS Console Minimized

Jan 17th, 2013

How to start an DOS console window minimized? (for an .bat or .cmd file) Here comes the code

@echo off

if "%1"=="start-min" goto :begin
start /min "" %0 start-min
exit

:begin
title Hello Mini World

echo Call something here

pause
exit
@echo on

这大概就是最直接，简单无需额外工具启动windows console并将其最小化的批处理脚本吧 :-p 非原创，网上捡来的。
