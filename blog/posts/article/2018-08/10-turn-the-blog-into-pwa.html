Turn the blog into PWA

Here’s comes my first online PWA: Progressive Web App

It’s a long way from playing Polymer 2.0 to using Polymer 3.0 beta to rebuild my very old blog here on the github pages.

I don’t like mass of dependencies from npm install. The pwa-starter-kit of polymer 3.0 requires lots of dev tools to help developing a WPA app from coding to publishing.

I’d tried building a docker images to keep all the resources to develop Polymer 3.0 apps. Cloning the mass packages inside node_modules/ to a new project when needed - to avoid the time-costing process of npm install.

After got a build demo, I try to get what’s the changed from the source code to the online version - and again, i could play the web development with just a simple editor.

Finally, the ~25MB node_modules works fine instead of using a 200+MB monster.

Nowdays, web is being much complex with more syntaxes and rules - TypeScript, React, Vue, Angular, Polymer, Dart …

Worse, there’s all kinds of build tools - WebPack, gulp, grunt, bower, npm, rollup …

Worst, I’m aging as web developer :D

This month, my work position is switched from web front-end developer to data analysist. I’d worked in learning Machine Learning more than last 20 months - Reading AI papers, using Spark, Kafka, Hive, HBase, PostgresQL, Docker - and programming in scala, python, golang …

The future is more complex and harder than I thought, I’m walking forward, make my challenges.

BTW, the PWA with polymer looks great. And I may try some Flutter apps later. And Expecting the new OS - Fuchsia from Google.
