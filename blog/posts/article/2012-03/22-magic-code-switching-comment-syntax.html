Magic Code Switching Comment Syntax

This a kind of c-style code comments to switch two blocks of code for debug or testing. 


/*/
//… Section A (commented)
/*/
//… Section B (working)
//*/

By adding one single / at the beginning, it turns into:


//*/
//... Section A (working)
/*/
//... Section B (commented)
//*/

Another simple section comment:


//*/
//... working
//*/

remove a single / at the beginning, it turns into:


/*/
//... commented
//*/
