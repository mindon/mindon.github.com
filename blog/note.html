Trouble/Solution

  • NodeJS – build 0.8 failed

/S/ 1. GNU Make 3.8.1+; 2. Python 2.6/2.7; 3. setenv FLOCK or export FLOCK; 4. setenv LINK=g++ or export LINK=g++

  • CSS – Text input component and button vertical align problem

/S/ It comes from the text vertical align, use table or change the padding-top to solve.

  • NodeJS – http client request POST, data lost in php or other CGI recipients

/S/ do NOT forget the headers with correct “Content-Length”, it matters.

  • Socket.IO – node: Memory Leak

/S/ skip websocket/flashsocket transports

  • NodeJS – node: error while loading shared libraries: libssl.so.1.0.0: cannot open shared object file: No such file or directory

/S/ >export LD_LIBRARY_PATH=$LD_LIBRARY_PATH:[include-libssl.so.1.0.0-path]

  • NodeJS – build 0.6.15 failed (make failure: undefined reference to `SSL_select_next_proto’)

/S/ update openssl to version 1.0.0+

  • NodeLoad – npm install nodeload, or manual download running problem under Windows

/S/ Problem comes from the log file name, it uses the time as part of file name. There’s character “:”, which is invalid as file name under Windows. replace this with a valid one.

  • NodeJS – configure fail, cannot find openssl in the path

/S/ config with “shared” option when building openssl. and configure with –openssl-* option to build node.

./config shared
    make
    make install
    ln -s /usr/lib/libssl.so.1.0.0 /usr/local/ssl/lib/libssl.so.1.0.0
    ln -s /usr/lib/libcrypt.so.1.0.0 /usr/local/ssl/lib/libcrypt.so.1.0.0
