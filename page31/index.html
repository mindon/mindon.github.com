巴士里的空气质量

BlogMS original blog key: 1001154858, blog id: airoff History stat: 浏览/评论：292/2 , 日期：2006年4月8日 20:46

我向来不喜欢封闭的空间，只因为空气不流通。所以我对空调也没什么特别的喜爱，我宁可使用风扇。但是对于封闭空间的担忧第一次因为巴士而变得激烈起来。我现在上下班坐的都是67路空调巴士，车内的空气一直特别稠闷，而且有时还充满了汽油味——浓得甚至让人担心会不会可能引起爆炸。

可能我们关注太多的都是一时的舒适，而不太担心一些长期的负面影响。就如空调巴士，给人舒适的只是温度上的，而对乘客健康影响更为直接的空气质量则成为被忽略的因素了。

CREATE FUNCTION more.firstPY(words varchar(255)) RETURNS mediumtext

BEGIN

declare fpy char(1);

declare pc char(1);

declare cc char(4);

set @fpy = UPPER(left(words,1));

set @pc = (CONVERT(@fpy USING gbk));

set @cc = hex(@pc);

if @cc >= "8140" and @cc <="FEA0" then

begin

select PY into @fpy from HZ2PY where hz>=@pc limit 1;

end;

elseif Instr("ABCDEFGHIJKLMNOPQRSTUVWXYZ0123456789",@fpy) = 0 then

set @fpy = "";

end if;

Return @fpy;

END
